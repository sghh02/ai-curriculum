# 付録: 実践ワークシート

この付録は、初心者でもそのまま使えるメモのテンプレート集です。
難しい言葉はできるだけ使わず、学習を始める前の整理とAIへの相談に使える形にしています。

## 1. テーマメモ

```text
1. 何を少し楽にしたいか
- 

2. 誰のため、または何のために取り組むか
- 

3. なぜそれに取り組みたいか
- 

4. 最初はどこまでできれば十分か
- 
```

## 2. 始める前の整理メモ

```text
1. 誰のため、または何のためか
- 

2. どんな場面で役立つか
- 

3. できたと言える状態
- 
- 
- 

4. 今回は作らないもの
- 
- 

5. まだ分からないこと
- 
```

## 3. AIと進め方を決めるメモ

```text
1. 最初に取り組むこと
- 

2. 最初の3ステップ
- 
- 
- 

3. 先に学ぶとよいこと
- 

4. 気をつけるポイント
- 

5. 試したあとに確かめること
- 
- 
```

## 4. AIに相談するときのテンプレート

```text
私は初心者です。
次の内容をもとに、最初に取り組む内容を小さく決めたいです。

テーマ:
[ここに書く]

始める前の整理:
[ここに書く]

教えてほしいこと:
1. 最初にやること
2. 今回はやらないほうがいいこと
3. 最初の3ステップ
4. 先に学ぶとよいこと
5. 進んだか確かめる方法
```

## 5. Gitの最小チェック

```text
- 作業前に git status を見た
- 変更後に git diff を見た
- add する前に差分を見た
- 小さいまとまりで commit した
- log で履歴を見た
```

## 6. 振り返りメモ

```text
1. 今回やったこと
- 

2. できるようになったこと
- 

3. 詰まったこと
- 

4. AIに聞いてよかったこと
- 

5. 次に試したいこと
- 

6. 次に学ぶこと
- 
```

## 7. 運営側の確認メモ

```text
- テーマが具体的か
- 最初の目標が小さいか
- AIへの質問が前に進む内容か
- 試した結果を確認できているか
- 振り返りで次の学習が見えているか
```
