# AI時代に腐らない「思考と要件整理」のカリキュラム

## 1. はじめに
AIモデルは数ヶ月単位で進化し、昨日まで使えていたツールや「魔法のプロンプト」はすぐに陳腐化します。
本カリキュラムは、特定のツールや小手先のテクニックを教えるものではありません。
時代が変わっても、どんなAIが登場しても一生使える**「人間の思考の型（要件整理）」と「AIを使い倒すマインドセット」**を身につけるための本質的なプログラムです。

## 2. 本カリキュラムのゴール
- **AIを「魔法の箱」ではなく、「思考を無限に拡張する最強のパートナー（カーナビ）」として認識する。**
- **「人間が要件（現在地と目的地）を整理し、AIと壁打ちして最適解（ルート）を導き出す」という自走サイクルを回せるようになる。**

## 3. カリキュラム構成
本カリキュラムは以下のステップで進行します。

1. **[第1章] マインドシフト** (`01_Mindshift.md`)
   - AIに対する誤解を解き、正しい向き合い方をインストールする。
2. **[第2章] 要件整理の極意** (`02_Requirement_Definition.md`)
   - AIに質の高いパスを出すための「人間の仕事（5W1H × As-Is/To-Be）」を学ぶ。
3. **[第3章] AIとの壁打ち・共創** (`03_Co-Creation_with_AI.md`)
   - 整理した要件をAIにぶつけ、思考を深める実践プロセス。
4. **[第4章] 無限の最適化と自走マインド** (`04_Self_Driving.md`)
   - 未知の課題に対して、AIを使って自ら学習・解決していく習慣をつくる。
5. **[付録] 実践ワークシート** (`05_Worksheets_Template.md`)
   - 日常業務ですぐに使える要件整理テンプレート。

さあ、AIに使われるのではなく、AIを使って自らの可能性を最大化する旅を始めましょう。
