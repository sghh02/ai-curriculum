# AI思考OSで学ぶ エンジニアへの進み方

## 1. このプログラムは誰のためのものか
このプログラムは、プログラミングが未経験、または少し触ったことがある人のためのものです。
「何から学べばいいか分からない」「AIは使ってみたけど学習にどう使えばいいか分からない」という状態から始められるように作っています。

## 2. 今までの学び方と、これからの学び方
今までのプログラミング学習は、`基礎 -> 応用 -> 実践` の順で進むことが多くありました。
まず文法を学び、次にフレームワークを学び、最後に作品を作る流れです。

このやり方は今でも間違いではありません。
ただ、AIがコードのたたき台を作れるようになった今、初心者にとってもっと進みやすい学び方があります。

それは、

`テーマを見つける -> 始める前に整理する -> AIに進め方を聞く -> 小さく試す -> 足りない基礎をその都度学ぶ`

という進め方です。

このプログラムは、こちらの進め方を採用します。

## 3. AI思考OSとは何か
AI思考OSは、AIにうまく頼りながら、自分で学習を進めるための考え方です。

難しいことをするためのものではありません。
次のことができるようになるための土台です。

- 自分は何をやりたいのか、何を学びたいのかを言葉にする
- それを始める前に整理する
- AIに何を聞けば前に進めるか考える
- 小さく作って、次に何を学ぶべきか決める

つまり、AI思考OSを身につけると、「今の自分には何が必要か」を自分で判断しやすくなります。

このプログラムでは、AI思考OSを次の3ステップで身につけます。

1. テーマを見つける
2. 始める前に整理する
3. AIと進め方を決める

## 4. このプログラムが目指すこと
このプログラムのゴールは、最初からすべてのコードを自力で書ける人にすることではありません。

目指すのは、次のような人です。

- 自分に合った学習テーマを小さく決められる
- AIに相談しながら学習の進め方を決められる
- 必要な基礎を必要な場面で学べる
- 学んだことを小さく試しながら前に進める
- 次に何を学べばよいかを自分で言える

## 5. このプログラムの進め方
このプログラムは、次の順で進みます。

1. テーマを見つける
2. 始める前に整理する
3. AIと一緒に進め方を決める
4. Gitで作業を残しながら小さく試す
5. 動かして確かめ、次の学習を決める

大事なのは、`全部学んでから作る` ではなく、`作りながら学ぶ` ことです。

## 6. このプログラムで大切にする考え方
- コードは目的ではなく、学びややりたいことを前に進める手段です
- AIは答えをもらう箱ではなく、考える相手です
- 分からないことは悪いことではなく、次に学ぶことのヒントです
- 最初から大きく作らず、小さい完成を目指します
- 動いたかだけでなく、思った通りに動いたかを確かめます

## 7. 受講後にできるようになること
このプログラムを終えると、次のような学び方ができるようになります。

- 学びたいことがぼんやりしていても、最初のテーマを自分で決められる
- 学習を始める前に、何を目指し、どこまでやるかを整理できる
- 詰まったときに、AIへ何を聞けば前に進めるか考えられる
- 分からないことを放置せず、次に学ぶべき内容として扱える
- 小さく試して、振り返りながら学習を続けられる
- 「今の自分に必要な学習」を自分で選べるようになる

## 8. プログラム構成
このプログラムは5章構成です。

- 第1章: `テーマを見つける`
  - 何をやりたいか、何を学びたいかを言葉にします。
- 第2章: `始める前に整理する`
  - 誰のためか、何を目指すか、どこまでできれば十分かを決めます。
- 第3章: `AIと進め方を決める`
  - AIに相談しながら、最初の小さい目標と進め方を決めます。
- 第4章: `Git / 作業を安全に残す`
  - 小さく試しながら進めるための最低限のGitを学びます。
- 第5章: `小さく試して学ぶ`
  - 実際に1つ試し、動作確認と振り返りまで行います。

## 9. このプログラムで残すもの
このプログラムでは、成果物だけでなく、次の学習記録も残します。

- テーマを書いたメモ
- 始める前の整理メモ
- AIに相談して決めた進め方
- Gitの作業履歴
- 動作確認メモ
- 振り返りメモ

## 10. 評価の考え方
評価は、「難しいものを扱えたか」ではありません。
次の点を見ます。

- 自分の学習テーマを具体的にできたか
- 最初の一歩を小さく決められたか
- AIに前に進むための質問ができたか
- 試したあとに振り返ることができたか
- 次に学ぶことを自分で決められたか

## 11. 受講者へのメッセージ
このプログラムは、初心者がいきなり専門用語を覚えるためのものではありません。
AIを使いながら、自分に合った進め方を見つけ、エンジニアになるための学習を自分で進められるようになるためのプログラムです。
