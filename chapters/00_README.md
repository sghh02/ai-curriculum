# AI時代に腐らない「思考と要件整理」のプログラム

## 1. はじめに
AIモデルは数ヶ月単位で進化し、昨日まで使えていたツールや「魔法のプロンプト」はすぐに陳腐化します。
本プログラムは、特定のツールや小手先のテクニックを教えるものではありません。
時代が変わっても、どんなAIが登場しても一生使える**「人間の思考の型（要件整理）」と「AIを使い倒すマインドセット」**を身につけるための本質的なプログラムです。

## 2. 対象読者
- AIツール（ChatGPT、Claudeなど）を使い始めたが、思うような回答が得られないと感じている方
- 「プロンプトの書き方」を調べてみたが、小手先のテクニックではなく本質を理解したい方
- 部下やチームにAI活用を推進したいが、何から教えればいいかわからないマネージャー

## 3. 本プログラムのゴール
- **AIを「魔法の箱」ではなく、「思考を無限に拡張する最強のパートナー（カーナビ）」として認識する。**
- **「人間が要件（現在地と目的地）を整理し、AIと壁打ちして最適解（ルート）を導き出す」という自走サイクルを回せるようになる。**

## 4. プログラム構成
本プログラムは以下のステップで進行します。

1. **[第1章] マインドシフト**
   - AIに対する誤解を解き、正しい向き合い方をインストールする。
2. **[第2章] 要件整理の極意**
   - AIに質の高いパスを出すための「人間の仕事（目的 → 現状と制約 → AIへの依頼）」を学ぶ。
3. **[第3章] AIとの壁打ち・共創**
   - 整理した要件をAIにぶつけ、思考を深める実践プロセス。
4. **[第4章] 無限の最適化と自走マインド**
   - 未知の課題に対して、AIを使って自ら学習・解決していく習慣をつくる。
5. **[付録] 実践ワークシート**
   - 日常業務ですぐに使える要件整理テンプレートと壁打ちフレーズ集。

## 5. よくある質問

**Q: 使うAIはChatGPTとClaude、どちらがいいですか？**
A: 本プログラムはツールを問いません。ChatGPT、Claude、Geminiなど、どのAIでも同じ原則が適用できます。大切なのはツールの選択ではなく、「要件を整理する思考の型」です。

**Q: 英語で入力したほうが精度は上がりますか？**
A: 日本語入力で十分です。回答精度を左右する本質的な要因は入力言語ではなく、「要件がどれだけ整理されているか」です。

**Q: どこまでAIに聞いていいかわかりません。**
A: 機密情報・個人情報を除けば、迷ったら積極的に試すのが一番の近道です。データ管理の注意点は第1章でカバーしています。
