# 第3章：AIとの壁打ち・共創プロセス（AIによる加速）

第2章で整理した「目的・現状と制約・AIへの依頼」をAIにぶつけ、実際に思考を加速させるプロセスを学びます。カーナビに例えるなら、現在地と目的地を入力した後、「渋滞を避けてルート再計算」を繰り返す工程です。

## 1. プロンプトの原石を投げる（ファーストコンタクト）
整理した要件をそのままAIに入力します。気の利いた「呪文」は不要です。
3ステップで整理した「目的」「現状と制約」「AIにやってほしいこと」をそのまま渡すだけで十分です。

## 2. 「発散」と「収束」の反復ループ
ここからがAIの真骨頂です。ラリーを通じてアイデアを磨き上げます。

- **【発散】圧倒的な量を出させる**
  - 「この要件を満たすアイデアを、突飛なものも含めてまずは30個出してください。」
  - 「別の切り口（例：コストゼロ、テクノロジー全振りなど）でさらに10個考えて。」
- **【収束】条件をつけて絞り込む**
  - 「出たアイデアの中から、実現可能性が高く、予算内に収まるトップ3を選び、その理由を解説して。」
  - 「アイデア2とアイデア5を組み合わせて、より強力な企画にブラッシュアップして。」
- **【検証】立場を変えて評価させる**
  - 「あなたは20代一人暮らしの社会人です。この企画に参加したいと思いますか？」
  - 「あなたは批判的な投資家です。この企画の最大のリスクを指摘してください。」

このように**AIに別の立場（ペルソナ）を与える**ことで、自分一人では気づけない視点を補えます。

### 実際の壁打ち例（冷凍食品の企画）

以下は、第2章の例を使って実際にAIとラリーする流れです。

> **あなた（1回目）：** この要件で企画アイデアを10個出して。
> *(3ステップで整理した要件を貼り付け)*
>
> **AI：** ①冷凍庫チャレンジTikTok ②一人暮らしあるあるリール ③冷凍食品アレンジレシピバトル ④…（10個のアイデア）
>
> **あなた（2回目・発散）：** いいね。もっと「バズりそう」な方向に振り切って、突飛なものも含めてさらに5個追加して。
>
> **AI：** ⑪冷凍食品だけで作るフルコースディナー選手権 ⑫「冷凍庫の中身公開」リレーチャレンジ…
>
> **あなた（3回目・収束）：** 全部の中から、予算ゼロで実現可能かつSNS拡散力が高そうなトップ3を選んで、理由も教えて。
>
> **AI：** トップ3は①③⑫です。理由：①は参加ハードルが低く…（具体的な根拠付き）
>
> **あなた（4回目・検証）：** あなたは20代一人暮らしの社会人です。トップ3に「参加したいか・シェアしたいか」を率直に評価して。
>
> **AI：** ①は面白そうだけど1週間は長い。3日チャレンジなら参加する。③はレシピバトルの審査基準がないと盛り上がりにくい…（具体的なフィードバック）

このように**発散→収束→検証**を繰り返すことで、最初の漠然としたアイデアが実行可能な企画に磨き上がっていきます。

### 壁打ちを終えるタイミング
ラリーをいつまでも続けるのではなく、以下のサインが出たら壁打ちを終了し、実行フェーズに移りましょう。

- 「次に誰が何をいつまでにやるか」が具体的に見えてきた
- 同じような回答が繰り返し返ってくるようになった（それ以上は深まらないサイン）
- 自分の中で「これで行こう」という判断が固まった

### うまくいかないときのサイン
ラリーが以下の状態になったら、第2章の3ステップに立ち戻りましょう。

- AIの回答がどんどん的外れになっていく → **目的か制約の伝え忘れがあるはず**
- アイデアが多すぎて収束できなくなった → **「どういう状態になっていれば最高か」という目的をより具体的に再入力する**
- AIの回答に引っ張られて当初の目的がずれてきた → **一度会話をリセットし、改めて3ステップから投げ直す**

## 3. 壁打ちの定番フレーズ集
「次に何を聞けばいいかわからない」というときは、以下のフレーズを使ってみましょう。

| やりたいこと | フレーズ例 |
|------------|----------|
| 深掘り | 「その中で最も効果が高いものはどれ？理由も教えて」 |
| 視点変更 | 「〇〇の立場だとこのアイデアはどう見える？」 |
| 具体化 | 「それを明日から始めるとしたら、最初の1歩は何？」 |
| 反論 | 「その案の一番の弱点は？どうすれば補える？」 |
| 組み合わせ | 「AとBのいいとこ取りをした案を作って」 |
| 条件変更 | 「予算が〇〇円あったらどう変わる？」 |
| 方向修正 | 「少し方向がずれてきた。もう一度〇〇という目的に立ち返って整理し直して」 |
| 制約の追加 | 「その提案は〇〇の制約に反している。その条件を守った上で再提案して」 |

**【実践のポイント】**
AIから返ってきた回答をそのまま鵜呑みにするのではなく、「ここは違う」「この制約を追加するのを忘れていた」と気づきを得て、**要件（目的・現状・制約）をアップデートしていくこと**が共創の最大の価値です。

**【ワーク】**
第2章のワークで整理した要件を使って、実際にAIと3往復の壁打ちをやってみましょう。発散・収束・検証のどれか1つでも試してみてください。

---
**【この章のまとめ】**
- 最初の一投は3ステップをそのまま貼るだけでいい。
- 発散→収束→検証のループでアイデアを磨く。立場を変えて評価させると死角が減る。
- 壁打ちは「次の質問」が命。定番フレーズを手元に置いておく。
- 行き詰まったら第2章の3ステップに立ち戻る。
