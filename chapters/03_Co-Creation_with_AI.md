# 第3章: AIと進め方を決める

## 1. この章でやること
ここでは、AIに「全部やって」と頼む前に、どう学び、どう進めるかを一緒に決めます。

初心者にとって大事なのは、最初の目標を小さくすることです。
この章では、最初にどこまでやるか、何から始めるか、何を先に学ぶかを決めます。

## 2. AIにいきなり答えを頼まない
AIは便利ですが、最初にいきなりコードや答えを出させると、分からないまま進みやすくなります。
先に次を相談したほうが進みやすいです。

1. 最初は何だけやるか
2. 何から手をつければいいか
3. どこでつまずきそうか
4. 何を確認すれば「できた」と言えるか

## 3. この章で決めること
AIと一緒に、次を決めます。

- 最初の小さい目標
- 最初の3ステップ
- 先に学ぶとよいこと
- 気をつけるポイント
- 試したあとに確かめること

## 4. AIへの聞き方
この章では、AIに進め方を教えてもらいます。

質問の例:

```text
以下の内容をもとに、初心者でも1週間で進められる形に小さくしてください。
そのうえで、
1. 最初にやること
2. 今回はやらないこと
3. 最初の3ステップ
4. 先に学ぶとよいこと
5. 進んだか確かめる方法
を分かりやすく教えてください。
```

```text
このテーマで最初につまずきやすい点を3つ挙げてください。
それぞれ、つまずいたときにAIにどう質問すればよいかも教えてください。
```

## 5. 良い進め方の形
良い進め方には、次の特徴があります。

- 最初の作業が軽い
- 途中で進歩が見える
- 分からない点がはっきりしている
- 次に聞くべき質問が見えている

## 6. この章のゴール
この章が終わったら、次がそろっている状態を目指します。

- 最初の小さい目標の説明
- 最初の3ステップ
- 先に学ぶことのメモ
- 確かめるポイント

**【ワーク】**
第2章で整理した内容をAIに見せて、「初心者向けの学習の進め方」に直してもらってください。
そのあと「もっと小さく始めるならどうする？」と聞いて、さらに1段階小さくしてください。

---
**【この章のまとめ】**
- AIには、まず答えではなく進め方を相談します。
- 大事なのは、最初の目標を小さく決めることです。
- AIに「何から始めればいいか」を聞けるようになると、自分で学習を進めやすくなります。
